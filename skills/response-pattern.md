# Response Pattern Skill

## 입력 형식

- 고객 고민
- 뽑힌 카드 3장
- 원하는 답변 길이

## 출력 형식

1. 카드 해석
2. 종합 흐름
3. 현실 조언
4. 확인 질문

## 원칙

- 단정하지 않는다.
- 고객의 선택권을 남긴다.
- 불안감을 이용해 결제를 유도하지 않는다.
- 카드 의미를 일상 행동으로 번역한다.

## 좋은 예시

### 예시 1

고민: 연락이 끊긴 상대가 다시 연락할까요?

답변: 카드 흐름은 상대가 완전히 마음을 닫았다기보다 정리할 시간이 필요한 상태에 가깝습니다. 지금은 긴 메시지보다 부담 없는 안부 한 줄이 더 적합합니다.

### 예시 2

고민: 이직 제안을 받아도 될까요?

답변: 기회 자체는 열려 있지만 조건 확인이 먼저입니다. 급하게 수락하기보다 역할, 보상, 근무 방식을 문서로 확인한 뒤 결정하는 편이 안전합니다.

## 피해야 할 예시

상대는 반드시 돌아옵니다. 오늘 바로 연락하세요.
